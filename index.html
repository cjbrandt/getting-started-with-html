Basic parts of the web

The web is an enormous network of computers that communicate with each other. The user accesses the web through a computer with a web browser. Through the internet the browser is able to obtain html documents from various servers using http. Those documents are then interpreted by the browser and displayed to the user in the manner which the programmers intended.

HTML

HTML is an acronym for HyperText Markup Language. It is the basis for everything on the web. HTML is made up of text content, which is "what you see" and markup, which is "how it looks."

HTML attributes - elements/tags

HTML documents consist of elements, which include various tags that tell the browser how to display the document.

Computers are stupid

Programmers like to say that computers are stupid because they do not think for themselves. They do exactly as they are told. So, any little typo or other minor error can cause a computer to fail to carry out the program. On the other hand, because they do exactly as commanded, this gives programmers a lot of power if they understand how to correctly provide the necessary commands.
